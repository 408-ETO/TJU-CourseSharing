# 概率论与数理统计 1

## 20211 学期 彭珍玲

讲教材大部分内容，包括随机事件与概率、随机变量及分布、多维随机变量及分布、随机变量的数字特征、大数定律和中心极限定理、数理统计的基本概念与抽样分布、参数估计、假设检验等。
